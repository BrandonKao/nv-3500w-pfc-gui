在線調整參數

已通過頁面授權，等待執行命令...

第一區塊：Adjust Mode / Data Control

第二區塊：Bus Voltage Adjust

輸入值 ×1000，uint16 LSB first。例：7.5 → 7500 → 0x1D4C → 4C 1D。

第三區塊：Xcap Offset Current

輸入值 ×1000，uint16 LSB first。例：0.75 → 750 → EE 02。

第四區塊：DCM Coefficient

輸入值 ×1000，uint16 LSB first。例：1.25 → 1250 → E2 04。

第五區塊：Voltage Loop

輸入值 ×100000，uint16 LSB first。Kp 與 Ki 分開輸入。

第六區塊：Current Loop

輸入值 ×100000，uint16 LSB first。Kp 與 Ki 分開輸入。
